Mis proyectos

- Motivar para vivir


Donación de página web a la asociación sin fines de lucro motivar para vivír, Esta pagina tiene como fin fomentar una cultura de prevención contra el cáncer y además, dar apoyo tanto moral como emocional a personas que padecen de esta enfermedad a través de artículos en su blog dónde se comparten hechos y experiencias reales de personas que atraviesan o superaron dicha enfermedad, dando así lugar a una maravillosa comunidad de apoyo colectivo. Cabe destacar que el visitante puede hacer una donación si así lo desea.


Tecnologías usadas:

Laravel, JavaScript, git.

- Empaque


Pequeña App web para empresa que permite gestionar sus recursos de forma mas eficiente, agregando registros detallados de las cajas a través del formulario e incluye un control de acceso para administrador, esto es especificamente para el control de la cantidad de cajas disponibles en stock. Actualmente en producción : Empaque


Tecnologías usadas:

PHP 7.4, Materialize CSS, AJAX, JavaScript, git.

- Mini tienda


Mini tienda de venta de libros digitales con carrito de la compra.


Tecnologías usadas:

PHP, Bootstrap, git

- Foro


Aplicación con autenticanción basado en tokens, Este foro permite la clasificación por categorías y con opción de clasificar los posts del mas reciente al mas antiguo, ademas el usuario puede marcar el post como pendiente o completado para su visualisación a la comunidad. Los integrantes de la comunidad pueden suscribirse o desuscribirse de algún post en especifico. Tambien cuenta con un campo de comentarios donde cualquier usuario registrado puede comentar y votar ( dar like ) a dicho post o comentario.


Tecnologías usadas:

Laravel with TDD, Git, Vuejs, Git.

- Blog


Creacion de un blog sencillo con laravel que incorpora un CRUD para la administracion de las publicaciones, además cada publicación esta relacionada con un categoría y con una etiqueta, la idea de este pequeño proyecto es comprender como funcionan las relaciones de datos con el ORM del framework.


Tecnologías usadas:

Laravel, Bootstrap, JavaScript.
